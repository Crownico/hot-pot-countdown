# 火锅计时器

辅助爱锅人士把握不同食材的下锅时间

### 简介

vue3 + typescript + vite + element-plus

### 功能

1. 添加食材进行计时
2. 添加或修改食材数据
3. 重置食材数据

### 安装与使用

- **Install：**

```text
pnpm install
```

- **Run：**

```text
pnpm dev
```

- **Build：**

```text
pnpm build
```
